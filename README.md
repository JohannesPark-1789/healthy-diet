# 🌿 항염증 식단 코치 (Anti-inflammatory Diet Coach)

40대 후반 한국 남성을 위한 **항염증·영양 밸런스 식단관리 PWA**.
근거 기반 딥리서치(지중해식·DII 식이염증지수·2025 한국인 영양섭취기준)를 한식 식단으로 옮긴 개인 맞춤 코치 앱.

## 기능

- **오늘** — 식사 기록 → 항염증 점수(0~100) 실시간 계산, 단백질·나트륨 게이지, 습관 체크(취침 3h 전 금식·매끼 단백질·금주), 체중·허리둘레 입력
- **추천** — 오늘 기록 기반 동적 추천("지금 채우면 좋아요": 단백질·등푸른생선 부족분) + 위장친화 메뉴 큐레이션
- **장보기** — 항염증 식단 기본 식재료 + 주간 부족분 힌트
- **추세** — 항염증 점수·단백질·체중·허리둘레 추이 그래프
- **설정** — 목표 편집, 저녁 리마인더, JSON 백업

## 기술

Vite + React + TypeScript + Tailwind CSS + vite-plugin-pwa.
모든 데이터는 **브라우저 로컬(localStorage)** 에만 저장 — 서버 없음, 개인정보 외부 전송 없음.

## 개발

```bash
npm install
npm run dev              # 로컬
npm run dev -- --host    # 같은 WiFi의 폰에서 접속
npm run build            # 프로덕션 빌드 (GitHub Pages용 base 적용)
```
